## 기본 문법

* !DOCTYPE 생성

  ```
  ! tab
  ```

* css 적용

  ```html
  <link rel="stylesheet" href="styles.css" />
  ```

* script 적용

  ```html	
  <script src="app.js"></script>
  ```

* 단축키

  ```html	
  <!--div.<class명>#<id명>>div.<class명>+div.<class명> TAB-->
  
  <div class="myclass" id="myid">
    <div class="newclass"></div>
    <div class="newclass"></div>
  </div>
  ```

* html에서 css 바로 적용

  ```html
   <div style="background-color: #ff3b30"></div>
  ```
